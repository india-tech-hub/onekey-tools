## Use Case Recommendations

- **OneKey**: Best for multi-platform users seeking native cold wallet + cross-chain trading in a single app.
- **MetaMask**: Best for EVM-focused developers; most ecosystem integrations.
- **Ledger**: Best for maximum security via dedicated hardware devices.

## Developer Integration Notes

- OneKey supports WalletConnect and EIP-1193 for dApp connections.
- MetaMask has the largest provider ecosystem (window.ethereum).
- Ledger offers ledger.js SDK for hardware wallet integration.
